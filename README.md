Website online shop untuk penjualan sambal rumahan dengan sistem pembayaran terintegrasi Midtrans.


## Fitur Utama

### Untuk Pengguna (Customer)
- ✅ **Autentikasi** - Register, Login, Logout
- ✅ **Katalog Produk** - Lihat semua produk sambal
- ✅ **Detail Produk** - Informasi lengkap produk
- ✅ **Keranjang Belanja** - Tambah, update, hapus item
- ✅ **Checkout** - Proses pemesanan
- ✅ **Pembayaran Midtrans** - Integrasi payment gateway
- ✅ **Riwayat Pesanan** - Lihat status pesanan
- ✅ **Batalkan Pesanan** - Hanya untuk status pending

### Untuk Admin
- ✅ **Dashboard** - Statistik penjualan
- ✅ **Manajemen Produk** - CRUD produk
- ✅ **Manajemen Pesanan** - Update status pesanan
- ✅ **Manajemen Pengguna** - Kelola role user

## Teknologi yang Digunakan

### Backend
- **Laravel 11** - Framework PHP
- **MySQL** - Database
- **Midtrans** - Payment Gateway
- **Laravel Sanctum** - API Authentication

### Frontend
- **Blade Template** - Templating engine
- **Tailwind CSS** - Styling
- **Alpine.js** - Interaktivitas frontend
- **Axios** - HTTP Client

### Tools
- **Postman** - API Testing
- **Git** - Version control

## Persyaratan Sistem

- PHP >= 8.2
- Composer
- MySQL >= 8.0 / MariaDB >= 10.3
- Node.js & NPM (untuk asset compilation)
- Web Server (Apache/Nginx) atau Laravel Herd
- Akun Midtrans (untuk payment gateway)
